# Business Operations & Growth Analytics

End-to-end analytics project demonstrating SQL, Python, and Power BI for business performance and growth insights.

## Project Overview
This project analyzes sales operations data to uncover:
- Revenue trends
- Regional performance
- Product-level insights
- Business growth patterns

The workflow simulates both small and large-scale datasets to reflect real-world analytics scenarios.

## Tech Stack
- **PostgreSQL** – Data storage & SQL analytics
- **Python (Pandas, Matplotlib)** – Data generation & validation
- **Power BI** – Interactive dashboard & business reporting
- **GitHub** – Version control & project showcase

## Project Structure
```
data/ → Raw & large datasets
notebooks/ → Python data generation & validation
sql/ → SQL analysis queries
powerbi/ → Power BI dashboard
```


## Key Insights
- Total Revenue & Order Volume
- Monthly revenue trends
- Revenue by region & category
- Top-performing products

## Dashboard Preview
The Power BI dashboard provides an executive-level view of business performance with interactive filters.
